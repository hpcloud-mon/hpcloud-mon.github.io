notifications

Notifications can be sent when alarm state transition occurs. Monasca supports an integrated notification engine that can send various notification types. Alarm Definitions can be associated with notification methods for transitions to the states, OK, Alarm or Undetermined.

Notification methods of the following types are supported:

  • Email
  • PagerDuty
  • Webhooks
