Monasca is an active OpenStack project and welcomes new development.

OpenStack Community

Monasca was formally accepted as an official OpenStack project in November, 2015 (https://review.openstack.org/#/c/213183/) and follows the OpenStack contribution guidelines and code standards.

Our project has a launchpad site which is used for bug tracking. The project mailing list is also hosted on launchpad, see the team page for details on that.

Details on the team meetings are found on this OpenStack wiki page.

Code Contributions

Monasca is hosted in github in the openstack organization and can be found here

The Monasca Vagrant environment is the primary setup used for development of Monasca. More information on testing Monasca can be found at our testing page
